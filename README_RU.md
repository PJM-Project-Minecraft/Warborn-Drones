# WRBDrones - Аддон дронов для SuperB Warfare

## Описание
WRBDrones - это аддон для мода SuperB Warfare, добавляющий новые типы дронов с уникальными характеристиками и ограничениями.

## Требования
- Minecraft 1.20.1
- Forge 47.4.0+
- SuperB Warfare 0.8.7.1+
- GeckoLib 4.4.6

## Типы дронов

### 1. FPV Дрон (Боевой)
**Характеристики:**
- Здоровье: 2.5 HP
- Размер: 0.65 x 0.32 блока
- Скорость: Высокая
- Маневренность: Высокая

**Разрешенные боеприпасы:**
- RPG Rocket (Standard) - стандартная ракета
- RPG Rocket (TBG) - термобарическая ракета
- Mortar Shell - минометный снаряд

**Применение:**
FPV дрон предназначен для точечных ударов по целям. Высокая скорость и маневренность позволяют быстро сблизиться с целью и нанести урон.

### 2. Mavic Дрон с модулем сброса (Разведка + Поддержка)
**Характеристики:**
- Здоровье: 2.5 HP
- Размер: 0.8 x 0.35 блока
- Скорость: Средняя
- Маневренность: Средняя

**Разрешенные предметы:**
- Medical Kit - медицинский набор (лечение)
- RGO Grenade - дымовая граната
- BLU-43 Mine - противопехотная мина

**Применение:**
Универсальный дрон для разведки и поддержки союзников. Может доставлять медикаменты, создавать дымовые завесы или устанавливать мины.

### 3. Mavic Дрон без модуля сброса (Только разведка)
**Характеристики:**
- Здоровье: 2.5 HP
- Размер: 0.8 x 0.35 блока
- Скорость: Средняя
- Маневренность: Средняя

**Ограничения:**
- НЕ может использовать боеприпасы
- НЕ может сбрасывать предметы
- Используется только для наблюдения

**Применение:**
Чисто разведывательный дрон для безопасного наблюдения за местностью без риска потерять ценные боеприпасы.

## Использование

### Создание дрона
1. Скрафтите предмет дрона (рецепты добавляются через datapacks)
2. ПКМ на землю или воду для размещения дрона
3. Дрон появится и будет готов к использованию

### Подключение монитора
1. Возьмите монитор из SuperB Warfare
2. ПКМ по дрону монитором
3. Монитор подключится к дрону
4. Используйте монитор (ПКМ) для управления дроном

### Загрузка боеприпасов
1. Возьмите разрешенный для данного типа дрона боеприпас
2. ПКМ по дрону боеприпасом
3. Боеприпас загрузится в дрон
4. Повторите для загрузки дополнительных боеприпасов (до максимума)

**Примечание:** При попытке загрузить неразрешенный боеприпас появится сообщение об ошибке.

### Разборка дрона
1. Присядьте (Shift)
2. ПКМ по дрону пустой рукой
3. Дрон разберется, вернув предмет дрона и загруженные боеприпасы
4. Все подключенные мониторы автоматически отключатся

## Особенности

### Анимация
- **Роторы** - вращаются, когда дрон в полете
- **Камера** - поворачивается вместе с взглядом игрока при управлении через монитор
- **Боеприпасы** - отображаются на модели дрона при загрузке

### HUD (Интерфейс)
Мод добавляет специальные HUD оверлеи для дронов:
- Основной HUD дрона (информация о состоянии)
- Специальный Mavic HUD (расширенная информация для Mavic дронов)
- Оверлей помех (при потере сигнала)

### Система ограничений
Каждый тип дрона имеет свой набор разрешенных боеприпасов:
- FPV: только боевые боеприпасы
- Mavic с модулем: поддержка и невзрывные предметы
- Mavic без модуля: ничего (только разведка)

Это позволяет сбалансировать игру и сделать каждый тип дрона уникальным.

## Интеграция с SuperB Warfare

Мод полностью интегрирован с SuperB Warfare:
- ✅ Использует систему мониторов SBW
- ✅ Совместим с боеприпасами SBW
- ✅ Работает с системой управления дронами SBW
- ✅ Поддерживает все механики дронов SBW

1. Звуковая система не реализована (используется система SBW)
2. Некоторые предупреждения компилятора (не критично)

### Добавление нового дрона

1. Создайте класс, наследующий `AddonDroneEntity`
2. Переопределите методы:
   - `getModelResource()` - путь к модели GeckoLib
   - `getTextureResource()` - путь к текстуре
   - `getAllowedAttachments()` - список разрешенных боеприпасов (или null для всех)
3. Зарегистрируйте тип в `ModEntityTypes`
4. Создайте предмет в `ModItems`
5. Зарегистрируйте рендерер в `ClientSetup`

Пример:
```java
public class MyDroneEntity extends AddonDroneEntity {
    private static final Set<String> ALLOWED_ATTACHMENTS = Set.of(
        "superbwarfare:my_ammo"
    );

    public MyDroneEntity(EntityType<? extends DroneEntity> type, Level level) {
        super(type, level);
    }

    @Override
    public ResourceLocation getModelResource() {
        return Wrbdrones.id("geo/my_drone.geo.json");
    }

    @Override
    public ResourceLocation getTextureResource() {
        return Wrbdrones.id("textures/entity/my_drone.png");
    }

    @Override
    protected Set<String> getAllowedAttachments() {
        return ALLOWED_ATTACHMENTS;
    }
}
```
